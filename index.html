Surf SMS Notifier

General

This project came about as I discovered that I had missed out on a day where we had awesome swell and didn't check the surf reports.

I tried to find an app that would notify me of good swell conditions based on swell, swell period, wind direction and swell size. None of the apps really met the requirements that I was after, so I set out to make my own.

This simple script works on a server that will ping the Surfline API for the conditions, it will check the conditions to see if they meet the parameters I set to what I beleive to be good surf conditions.

If the surf conditons meet my parameters it will then put this information into a simple text message that it then utilises the Twilio API to send me a notification the night before to let me know the conditojns are going to be good.
